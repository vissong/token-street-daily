# AI 日报 · 词元长街

Generated by the `ai-newsletter-builder` skill.

- `index.html` — homepage, regenerated by `scripts/build_index.py`.
- `issues/<date>.html` — one file per daily issue.
- `data/issues.json` — lightweight manifest used by the homepage.
- `config/sources.yaml` — data source definitions.
- `config/design.md` — active design system.

## Rebuild the homepage

```bash
python <path-to-skill>/scripts/build_index.py --site .
```
